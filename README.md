# B3 Agent · Rico (XP)

Monitor de investimentos com cotações em tempo real, alertas e assistente IA.

## Como usar

### Windows
1. Clique duas vezes em **launch.bat**
2. O navegador abre em `http://localhost:8741`
3. No Chrome/Edge: clique no ícone **⊕ Instalar** na barra de endereços

### Mac / Linux
```bash
bash launch.sh
```

### Requisitos
- Python 3 **ou** Node.js instalado
- Chrome, Edge ou outro navegador moderno

## Instalar como app (PWA)
Após abrir via `launch.bat`, no Chrome/Edge:
- Barra de endereços → ícone **⊕** → **Instalar B3 Agent**
- O app fica no menu Iniciar/Dock como qualquer programa
